### 数据转换说明

1. **ID前缀**：所有ID添加了 `ds_` 前缀，避免与其他科目冲突
   - 章节ID: `01` → `ds_01`
   - 题目ID: `q011002` → `ds_q011002`
   - 知识点ID: `kc0111` → `ds_kc0111`

2. **数据格式转换**：
   - 添加了 `subject: "data_structure"` 字段
   - 转换了难度格式（字符串 → 数字）
   - 转换了题目类型
   - 添加了标签 `["数据结构", "计算机科学"]`

3. **保留原有数据**：
   - 原有 `data/ds_data/` 目录保持不变
   - 可以作为参考或备用

## 📝 使用方法

### 在前端查看数据结构题目

1. 访问 `http://localhost:3003/problems`
2. 在科目选择器中选择 **"数据结构"**
3. 选择章节查看题目
4. 点击题目开始学习

### 重新生成索引

如果数据有更新，需要重新生成索引：

```bash
python data/high_school_data_loader.py
```

或者删除索引文件，系统会自动重建：

```bash
rm data/hs_indices.pkl
```
